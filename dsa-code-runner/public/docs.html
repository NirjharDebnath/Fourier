How to Use

This platform is designed to be simple and intuitive. Follow the steps below to get started.

For Students

  1. From the home page, click on the Student Login button.
  2. Enter your full name and university roll number to access the IDE.
  3. Once in the IDE, select your programming language from the dropdown. The editor will provide a "Hello World" snippet.
  4. Write your code in the professional-grade Monaco Editor (the engine behind VS Code), which provides syntax highlighting.
  5. Provide any necessary standard input (stdin) in the right text area.
  6. Click Run Code to execute your program. The output, including any text or images, will appear below.

For Teachers

  1. From the home page, click on the Teacher Login button.
  2. Enter the admin credentials (default: admin / password) to access the dashboard.
  3. The dashboard provides a live, read-only view of all currently logged-in students.

Key Features

Multi-Language Support

The runner supports the following languages with these specific requirements:

  • Python: Uses Python 3.9. Pre-installed libraries include numpy, pandas, scipy, scikit-learn, sympy, and matplotlib.
  • C/C++: Uses the latest GCC/G++ compilers available in the Debian Bullseye repository.
  • Java: Uses OpenJDK 17. The system automatically detects your public class name, so you are not required to name your class Main.

Graphical Plots with Matplotlib

You can generate and view plots from your Python code. Instead of using plt.show(), which will not work, save your plot to a file named output.png. The system will automatically detect this file and display the image in the output box.

Example: plt.savefig("output.png")

Security and Isolation

Every code submission is executed inside a brand new, isolated Docker container. This container has no network access and has strict memory and CPU limits, ensuring that code runs securely without affecting the server or other users. The container is immediately destroyed after execution.

Technology Stack

  • Backend: Flask (Python Web Framework)
  • Containerization: Docker
  • Frontend: HTML, Tailwind CSS, JavaScript
  • Code Editor: Monaco Editor
