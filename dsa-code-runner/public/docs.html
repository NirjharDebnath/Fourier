How to Use

This platform is designed to be simple and intuitive. Follow the steps below to get started.

For Students

  1. From the home page, click on the Student Login button.
  2. Enter your full name and university roll number to access the IDE.
  3. Once in the IDE, select your desired programming language from the dropdown menu.
  4. Write your code in the left text area and provide any necessary standard input (stdin) in the right text area.
  5. Click the Run Code button to execute your program in a secure Docker container. The output will appear at the bottom.

For Teachers

  1. From the home page, click on the Teacher Login button.
  2. Enter the admin credentials to access the dashboard.
  3. The dashboard provides a live view of all currently logged-in students, including their roll number, name, IP address, and login time.

Supported Languages

The runner supports the following languages with these specific requirements:

  • Python: Uses Python 3.12.
  • C/C++: Uses the latest GCC/G++ compilers.
  • Java: Uses OpenJDK 17. The system is smart! It will automatically detect the name of your public class and name the file accordingly, so you don't have to name it Main.
